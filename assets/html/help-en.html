How to use Ice Hockey Scoresheet

To close this help screen, use the Android back button.

Basics

The game is recorded as a series of events. There are several types of event…

* Goal scored (home or away)
* Penalty (home or away)
* End of period

Every event includes the time that it happens; this can be either the clock time within a period (which counts down within the period) or the game time, which counts up from the start of the first period. You record new events using the period and clock time, and they are automatically converted into game time for you.

Once you have recorded the events for the game, the Game Report will display all the details in the format required by the EIHA official score sheet.

Important - make sure you write down the game events on a piece of paper as well as recording them in the scoresheet app. If the app crashes and loses the entire game record you may need to fill in the official scoresheet manually.

User Interface Layout

The scoreboard is shown at the top of the screen, with the home and away goal totals, and the current period. Above the scoreboard is the title bar, which includes the options menu. The options menu lets you start a new game, save or load games, etc.

Below the scoreboard you will normally see the Game History, a list of all the events recorded so far for the game. Between the scoreboard and the events list are a set of buttons for recording new events. Also above the events list is the Game Report button which is used to access the game report, with all the information needed to fill in the official scoresheet.

Adding Events

Events are added to the game using the five new event buttons.

* HOME GOAL
* HOME PENALTY
* NEXT (period)
* AWAY GOAL
* AWAY PENALTY

When adding goal or penalty events you will be prompted to enter additional information.

The correct period should have been identified, but you can change it if necessary.

The game clock time should be entered as it appears on the rink's scoreboard, as a four digit number with no punctuation. For example, if the clock shows 15 minutes 27 seconds left in the period, you should enter "1527". If the clock shows 9:32 remaining, enter "0932" and if there are only 23 seconds left in the period then enter "0023".

The clock time will be automatically converted to game time (since the start of period 1) when the event is added to the game history.

Adding Goals

The default goal type is "E" (the code for "Even handed") but you can change this. As you start to type, valid options (E, SH, PP, DP, PS, PG, ENG) will be presented for you to select.

You must enter the player number for the goal scorer. You can also add player numbers for either one or two assists.

Adding Penalties

The default for penalty minutes is 2, but this can be changed as appropriate.

Penalty codes can be entered, and as with goal type valid codes are shown as you type. You can generally just type the first letter or two of the penalty code and then select the correct value from the list. See the end of this help text for a list of valid penalty codes.

You must enter the number of the player receiving the penalty.

Note that the time recorded for the penalty will be used for both the awarded time and the start time; delayed penalties are not currently supported. The end time is calculated automatically.

The following penalty codes are valid...

ABUSE, ADJ-E, ATT-B, ATT-K, ATT-S, ALTRC, BENCH, BDYCH, BOARD, BR-ST, BUT-E, CHARG, CHE-B, CLIP, CROSS, DAN-E, DELAY, DIS-N, ELBOW, EX-RP, F-OFF, FAL-P, FIGHT, GA-MI, GR-MI, HANDP, H-BUT, HI-ST, HOLD, HO-MA, HO-ST, HOOK, IL-EQ, IL-ST, INTRF, INT-S, KICK, KNEE, L-CRE, L-PEB, L-PLB, MATCH, MISC, MOLST, OBS-L, PEN-S, P-GRE, P-HOL, P-OUT, REFUS, ROUGH, SLASH, SPEAR, STA-L, THR-P, THR-S, TOO-M, TRIP, UN-SP, UNN-R

Deleting or changing events

You can delete events by holding down on them for a second or two in the game history, and then selecting "Yes" at the prompt.

You cannot currently edit and existing event. If you get an event wrong, just delete it and create it again. This works even if the event is not in the current period, as long as you specify the change the period for the event when (re)creating it.

Game Report

The Game Report shows all the information for the game in the format required by the EIHA score sheet...

* Home player stats (goals/assists/penalty minutes)
* Home scoring
* Home penalties
* Away player stats
* Away scoring
* Away penalties
* Period scores
* Period penalty minutes

You can simply copy the information from the game report screen into the written score sheet.

All clock conversions and calculations are performed automatically from the game events.

Saving and Loading

Saving

Select Save Game from the options menu to save the current game. Game files are saved in the "Documents\Scoresheets" folder. You do not need to specify a file name as they are named automatically based on the current date and time, plus the current game period and score.

Loading and Managing Saved Game File

Selecting Load Game from the options menu displays a list of all the saved game files, allowing you to load them back into the app.

You can easily load the last saved file which will be called gamedata.json, but you can also load earlier save files. Each saved file includes the date and time it was created, plus the period and home and away scores, allowing you to identify the best file to load.

Toggling the All files switch will show or hide all these automatically-named files.

While a quick tap is enough to load a file, a long tap will open a menu giving you the option to load, delete or rename the file. After you rename a file it will be included in the list when you toggle "all files" off.

Game Details

Selecting Game Details from the options menu allows you to record the team names and the venue/location for the game.

Miscellaneous

If the app gets "confused" and doesn't appear to display correctly, try the "Refresh Screen" option from the options menu.
